PlantUML Server

Encoding URL for PlantUML Server for source and diagram
=========================================================
A PlantUML server generates the output image from input PlantUML source code.

It is possible to setup your own server (docker images available).

Here we will use the public plantuml server http://www.plantuml.com/plantuml where we can enter our PlantUML source code.


We will use our :ref:`Context` diagram source as an example.

.. literalinclude:: ./plantuml/C4-PlantUML/examples/C4_ContextDiagramSample-bigbankplc-landscape.puml
    :linenos:
